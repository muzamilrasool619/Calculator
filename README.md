## Deploy to GitHub Pages

1. Create a GitHub repository and push this project to the `main` branch.
2. The workflow `.github/workflows/deploy.yml` (added to this repo) will publish the site to the `gh-pages` branch automatically on push.
3. Check Actions → the `Deploy` workflow run; then enable Pages in Settings if required. Your site will be at `https://<your-username>.github.io/<repo>/`.

If you prefer manual setup, enable GitHub Pages in the repository Settings and choose `gh-pages` as the source.
